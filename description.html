Chapter 2 Description

This is an R bookdown project containing a reporting approach for displaying project risk analysis data to the BRIP stakeholders. This tool is built based on USACE best practice documented in the USACE Planning Community Toolbox, Risk Informed Decision Making and Planning Support Tools website.

2.1 Funding

Funding for development and maintenance of this reporting tool has been provided by the following US Army Corps of Engineers (USACE) projects:

  • Brandon Road Interbasin Project (BRIP).
